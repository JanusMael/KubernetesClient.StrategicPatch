# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

Instances of unacceptable behavior may be reported to the project maintainer,
Brian Bennewitz, at **bbennewitz@gmail.com**. All complaints will be reviewed
promptly and handled with discretion.
